Equipments

Stabiliser

A stabiliser allowing me to capture smooth, professional-looking footage even when I’m on the move. This project cannot be done without this equipment, this is to ensure smooth footages and maintained story telling throughout the video. Smooth footage gives videos a polished and professional appearance. It helps convey a sense of quality and competence in filmmaking.

  • Focus on content:
    • When the technical aspects of footage are smooth, viewers are less likely to be distracted, allowing them to focus more on the content, story, or message being conveyed
  • Versatility in Editing:
    • Smooth footage is more versatile during the editing process. It allows for seamless transitions, smoother cuts, and better integration of various shots.
  • Cinematic Look:
    • It adds a level of sophistication and artistry to the visuals, contributing to the overall storytelling and mood.
  • Viewer Comfort:
    • Shaky or jittery footage can be uncomfortable for viewers to watch. Smooth footage ensures a more enjoyable and immersive viewing experience.

Camera & Lenses

Advanced cameras and lenses offer a range of features and capabilities, This is a full-frame camera, so it has a larger sensors, leading to better image quality with improved dynamic range, low-light performance, and reduced noise.

    The 2 lenses:

  • 24-70mm
    • A 24-70mm zoom lens covers a wide range of focal lengths, from wide-angle to short telephoto. This means that you can capture a variety of subjects and scenes without having to change lenses, making it a great all-around lens for everyday video.
    It’s a versatile and powerful tool that can be used to capture a wide variety of images, it can replace multiple prime lenses, offering convenience by eliminating the need to switch lenses frequently. This is particularly beneficial in situations where changing lenses quickly is challenging or impractical.
  • 16mm
    • A wide-angle lens allows me to capture a lot of the scene in a single frame, which can be useful for landscapes. Due to its short focal length, a 16mm lens has a greater depth of field than longer focal lengths. This means that more of the scene will be in focus, which can be helpful for landscape.

    Variable ND Filter and CPL

    In changing light conditions, adjusting the variable ND filter allows you to maintain a consistent exposure without having to constantly change your camera settings. This is especially helpful for video recording or situations where changing settings would be impractical.
    This specific ND filter allows me to reduce the amount of light entering your camera by 2 to 8 f-stops. This is ideal for situations like bright sunlight when you want to use a slower shutter speed to achieve creative effects like silky water or motion blur (useful especially I’m filming rivers mostly).
  • Maintain consistent exposure in changing light conditions or to achieve cinematic motion blur

    • Reduce glare and reflections:

    This is the most common use of a CPL filter. It effectively cuts through glare and reflections from surfaces like water, glass, and metal. A CPL filter can make colours appear more saturated and vibrant. This is especially noticeable in blue skies, green foliage, and other natural elements. The polarizing effect of the filter can help to remove atmospheric haze and improve the overall clarity of my footages. This is particularly useful for capturing distant landscapes or improving the visibility of details in scenes with a lot of air pollution.

Website

The first thing to do when creating a website is to define structure for it:

  • HTML is the essential code that tells the browser what to display and how to arrange it.
  • HTML elements define the different sections and components of a web page, making it easy for users to navigate and understand the content.
  • CSS and JavaScript rely on HTML to apply styles and interactivity to the website's content, along with Bootstrap library to create responsive experience.

Then there is CSS, this is where I choose to style my website.

  • CSS describes how HTML elements should be displayed, governing their visual properties like font, colour, background, layout, and animations.
  • CSS separates presentation from content, allowing us to modify the visual appearance of your website without affecting the underlying content defined in HTML. This promotes code organization and reuse.
  • CSS can improve the usability of your website by making it easier for users to find the information they need by emphasizing important content and creating clear visual cues.

JavaScript is a programming language that is essential for many websites to function properly. For this website, Javascript is the driving force behind interactive elements:

  • JavaScript is the essential ingredient that injects interactivity and dynamism into websites. While HTML defines the structure and CSS governs the appearance, JavaScript adds the lifeblood of responsiveness and user engagement.
  • JavaScript is a scripting language that adds dynamic behaviour to web pages. It allows us to respond to user interactions, manipulate the content and create complex animations and applications.
  • JavaScript is the driving force behind interactive elements like drop-down menus, image galleries, and real-time data updates, making websites more engaging and user-friendly. Here is where my design comes form.
